# Todo App Overview

## Purpose
A todo application that evolves from console app to AI chatbot.

## Current Phase
Phase II: Full-Stack Web Application

## Tech Stack
- Frontend: Next.js 16+, TypeScript, Tailwind CSS
- Backend: FastAPI, SQLModel, Neon PostgreSQL
- Auth: Better Auth with JWT

## Features
- [x] Task CRUD operations (Phase I)
- [ ] User authentication (Phase II)
- [ ] Task filtering and sorting (Phase II)
- [ ] Multi-user support with data isolation (Phase II)
